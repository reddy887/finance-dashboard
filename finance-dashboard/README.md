# Personal Finance Dashboard

This is a frontend finance dashboard built using React.

## Features

- View income, expenses, and balance
- Interactive charts (line + pie)
- Transaction management (add/edit)
- Role-based UI (Admin / Viewer)
- Budget tracking with progress bar
- Insights on spending patterns

## Tech Stack

- React
- Tailwind CSS
- Recharts

## Notes

This project uses mock data and localStorage for persistence.
